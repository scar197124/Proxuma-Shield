# Security Policy

## Supported Versions
RazorPrime v1 (current release) receives:
- UI corrections
- Scanner accuracy improvements
- False‑positive/false‑negative tuning

## Reporting a Vulnerability
If you discover a bug, false reading, or security issue:
Open an issue on GitHub with description + reproduction steps.

## Data Security
Proxuma Shield performs **all scans locally in the browser**.
No data, URLs, or results are sent to any server.
